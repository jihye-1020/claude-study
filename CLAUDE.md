# 나는 누구인가
트립스토어의 CRM 마케터, 비개발자

# 내가 자주 하는 업무
- CRM 발송 데이터 기입
- 푸시메세지 문구 기획
- Amplitude/BigQuery 기반 데이터 분석

# 결과물 형식
- 마크다운 형식으로 작성
- 정확한 데이터 기반으로 임의 해석을 덧붙이지 않은 수치만 제시
- 핵심만 간단히 요약

# 금지 사항
- 임의로 수치를 기입하는 것
- 작성자의 말이 무조건 옳다는 전제

# 자주 쓰는 맥락
- 데이터 분석 결과는 무조건 분석가 서브 에이전트를 활용해서 검증 후 제시

# 추가 컨텍스트
- CRM 발송 채널: 앱 푸시메세지, 카카오 브랜드메세지
- 주요 분석 지표 우선순위: CAC > 전환율 > 클릭율
- 사용 툴: 푸시메세지 발송(Braze), 카카오 플친메세지 발송(Bigin), 데이터 분석(BigQuery, Amplitude)
- 주요 분석 대상: 캠페인 발송 데이터, 자동화 메세지 데이터, 유저 전환 데이터
- Claude 주요 활용 목적: 데이터 해석, CRM 발송 자동화를 위한 작업
